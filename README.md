# Chicago Crime Analysis

## Overview

This project analyzes Chicago crime data to identify crime hotspots, examine school-related incidents, and explore the relationship between community socioeconomic conditions and crime.

SQL was used to query and aggregate the data, while Python was used to create visualizations and communicate key findings.

---

## Objectives

- Identify crime hotspots across Chicago communities.
- Examine crimes occurring at schools.
- Explore the relationship between socioeconomic indicators and crime.

## Key Findings

- Community Area **25** had the highest number of reported crimes.
- **Battery** was the most common school-related crime.
- Communities with higher hardship indexes showed a **weak positive correlation** with crime count.
- Per capita income showed **little to no linear relationship** with crime count.
- Poverty rate had the strongest relationship with crime count among the socioeconomic indicators analyzed, though the correlation remained weak.
